## Key Concepts Explanation - Vysvƒõtlen√≠ Kl√≠ƒçov√Ωch Koncept≈Ø

### Multi-Agent Systems (Syst√©my V√≠ce Agent≈Ø)
Architektura, kde nƒõkolik autonomn√≠ch AI agent≈Ø spolupracuje na spoleƒçn√©m c√≠li. Ka≈æd√Ω agent m√° specifickou roli a m≈Ø≈æe komunikovat s ostatn√≠mi agenty.

### Procedural Content Generation (Procedur√°ln√≠ Generov√°n√≠ Obsahu)
Algoritmick√© vytv√°≈ôen√≠ hern√≠ho obsahu (√∫rovnƒõ, p≈ô√≠bƒõhy, postavy) nam√≠sto ruƒçn√≠ho navrhov√°n√≠. Umo≈æ≈àuje nekoneƒçnou variabilitu a personalizaci.

### LLM Agents (LLM Agenti)
Specializovan√© AI agenty postaven√© na velk√Ωch jazykov√Ωch modelech, ka≈æd√Ω optimalizovan√Ω pro konkr√©tn√≠ √∫koly jako psan√≠, design nebo testov√°n√≠.

### Collaborative Creativity (Kolaborativn√≠ Kreativita)
Proces, kde AI agenti spoleƒçnƒõ vytv√°≈ôej√≠ kreativn√≠ obsah, kombinuj√≠ r≈Øzn√© perspektivy a expert√≠zy pro dosa≈æen√≠ lep≈°√≠ch v√Ωsledk≈Ø.

## Comprehensive Project Explanation - Komplexn√≠ Vysvƒõtlen√≠ Projektu

### C√≠l Projektu
Vytvo≈ôen√≠ autonomn√≠ho kolektivu AI agent≈Ø, kte≈ô√≠ spolupracuj√≠ na designu a v√Ωvoji hern√≠ho obsahu. Syst√©m kombinuje r≈Øzn√© specialized agenty - hern√≠ design√©ry, scen√°risty, balanc testery a narrative architekty.

### Hlavn√≠ V√Ωzvy

- **Kvalita obsahu**: Udr≈æen√≠ vysok√© kreativity p≈ôi zachov√°n√≠ hern√≠ logiky
- **Iterativn√≠ zlep≈°ov√°n√≠**: Implementace feedback loops pro kontinu√°ln√≠ optimalizaci
- **Konzistence**: Zaji≈°tƒõn√≠ koherentnosti nap≈ô√≠ƒç r≈Øzn√Ωmi elementy hry

### Potenci√°ln√≠ Dopad
Revolucionizace hern√≠ho pr≈Ømyslu prost≈ôednictv√≠m:
- Dramatick√©ho sn√≠≈æen√≠ ƒçasu v√Ωvoje
- Personalizovan√©ho obsahu pro jednotliv√© hr√°ƒçe
- Nekoneƒçn√© variability hern√≠ch z√°≈æitk≈Ø

class GameGenre(Enum):

    def collaborative_iteration(self,

                3. Zachov√°v√° pozitivn√≠ aspekty

                Zamƒõ≈ôte se na celkov√© zlep≈°en√≠ hern√≠ho z√°≈æitku.

            crew = Crew(

        except Exception as e:

    def generate_complete_game_design(self, 
                                    genre: GameGenre, 
                                    target_audience: str, 
                                    theme: str) -> Dict[str, Any]:
        """Kompletn√≠ workflow pro generov√°n√≠ hern√≠ho designu"""
        
        logger.info("Spou≈°t√≠m kompletn√≠ workflow pro generov√°n√≠ hern√≠ho designu")
        
        try:
            # 1. Generov√°n√≠ z√°kladn√≠ho konceptu
            game_concept = self.generate_game_concept(genre, target_audience, theme)
            
            # 2. Generov√°n√≠ narrativn√≠ho obsahu
            narrative_content = self.generate_narrative_content(game_concept)
            
            # 3. Generov√°n√≠ hern√≠ch mechanik
            mechanics_content = self.generate_game_mechanics(game_concept)
            
            # 4. Testov√°n√≠ balance
            narrative_feedback = self.test_content_balance(narrative_content)
            mechanics_feedback = self.test_content_balance(mechanics_content)
            
            # 5. Iterativn√≠ zlep≈°ov√°n√≠
            improved_mechanics = self.collaborative_iteration(game_concept, mechanics_feedback)
            
            # 6. Fin√°ln√≠ kompilace
            complete_design = {
                "game_concept": {
                    "title": game_concept.title,
                    "genre": game_concept.genre.value,
                    "target_audience": game_concept.target_audience,
                    "theme": game_concept.theme,
                    "estimated_playtime": game_concept.estimated_playtime
                },
                "narrative": {
                    "content": narrative_content.dict(),
                    "feedback": narrative_feedback.dict()
                },
                "mechanics": {
                    "original": mechanics_content.dict(),
                    "improved": improved_mechanics.dict(),
                    "feedback": mechanics_feedback.dict()
                },
                "overall_quality": {
                    "content_pieces": len(self.content_database),
                    "average_quality": sum(c.quality_score for c in self.content_database) / len(self.content_database),
                    "iterations_performed": 1
                }
            }
            
            logger.info("Kompletn√≠ hern√≠ design √∫spƒõ≈°nƒõ vygenerov√°n")
            return complete_design
            
        except Exception as e:
            logger.error(f"Chyba p≈ôi generov√°n√≠ kompletn√≠ho hern√≠ho designu: {e}")
            raise

    def export_design_document(self, design: Dict[str, Any], filename: str):

            logger.info(f"Hern√≠ design exportov√°n do {filename}")
            
        except Exception as e:
            logger.error(f"Chyba p≈ôi exportu designu: {e}")
            raise

    def get_analytics_report(self) -> Dict[str, Any]:
        """Generuje analytickou zpr√°vu o v√Ωkonu kolektivu"""
        
        if not self.content_database:
            return {"message": "≈Ω√°dn√Ω obsah k anal√Ωze"}
        
        content_by_type = {}
        for content in self.content_database:
            content_type = content.content_type.value
            if content_type not in content_by_type:
                content_by_type[content_type] = []
            content_by_type[content_type].append(content.quality_score)
        
        analytics = {
            "total_content_pieces": len(self.content_database),
            "average_quality": sum(c.quality_score for c in self.content_database) / len(self.content_database),
            "quality_by_type": {
                content_type: {
                    "count": len(scores),
                    "average_quality": sum(scores) / len(scores),
                    "max_quality": max(scores),
                    "min_quality": min(scores)
                }
                for content_type, scores in content_by_type.items()
            },
            "feedback_summary": {
                "total_feedback_reports": len(self.feedback_history),
                "average_balance_score": sum(f.balance_score for f in self.feedback_history) / len(self.feedback_history) if self.feedback_history else 0,
                "average_fun_factor": sum(f.fun_factor for f in self.feedback_history) / len(self.feedback_history) if self.feedback_history else 0
            }
        }
        
        return analytics

from typing import Dict, List, Any, Optional

class DifficultyLevel(Enum):

    """Datab√°ze hern√≠ch designov√Ωch prvk≈Ø"""
    
    def __init__(self):
        self.levels: List[GameLevel] = []
        self.characters: List[Character] = []
        self.items: List[GameItem] = []
        self.design_patterns: Dict[str, Any] = {}
    
    def add_level(self, level: GameLevel):
        """P≈ôid√° novou √∫rove≈à do datab√°ze"""
        self.levels.append(level)
    
    def add_character(self, character: Character):

        """P≈ôid√° nov√Ω p≈ôedmƒõt do datab√°ze"""
        self.items.append(item)

        """Vr√°t√≠ √∫rovnƒõ podle obt√≠≈ænosti"""

            "exported_at": datetime.now().isoformat()

        with open(filename, 'w', encoding='utf-8') as f:
            json.dump(data, f, ensure_ascii=False, indent=2, default=str)
