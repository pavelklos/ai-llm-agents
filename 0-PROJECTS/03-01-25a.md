## Project Title

**AI-Powered Art History and Museum Guide with Retrieval-Augmented Generation**

## Key Concepts Explanation

### **RAG (Retrieval-Augmented Generation)**
A hybrid AI approach that combines information retrieval from comprehensive art databases with generative AI to provide contextual information about artworks, artists, and cultural movements, creating personalized and educational museum experiences.

### **Artwork Information**
Comprehensive metadata about art pieces including title, medium, dimensions, creation date, provenance, current location, and detailed descriptions that form the foundation of curatorial knowledge systems.

### **Artist Biographies**
Detailed life stories, artistic development, influences, and cultural significance of artists throughout history, providing context for understanding their works and contributions to art movements.

### **Exhibition Catalogs**
Scholarly publications documenting art exhibitions, including critical essays, artwork descriptions, and curatorial insights that provide academic depth to art historical understanding.

### **Cultural Context**
Historical, social, and political circumstances surrounding artwork creation, including artistic movements, cultural influences, and societal factors that shaped artistic expression.

### **Museum APIs**
RESTful web services providing programmatic access to museum collections, artwork metadata, high-resolution images, and exhibition information from institutions worldwide.

### **Image Analysis**
Computer vision and machine learning techniques applied to artwork images for style recognition, period identification, and visual similarity matching to enhance discovery and understanding.

### **Virtual Tours**
Interactive digital experiences that simulate physical museum visits, allowing remote exploration of collections with contextual information and guided narratives.

## Comprehensive Project Explanation

The Art History and Museum Guide represents an innovative AI-driven platform that transforms how people interact with and learn about art. This system leverages RAG architecture to combine extensive art historical databases with intelligent analysis capabilities, creating personalized educational experiences that make art more accessible and engaging for diverse audiences.

### **Objectives**

1. **Enhanced Art Education**: Provide comprehensive, contextual information about artworks and artists that adapts to user knowledge levels and interests.

2. **Intelligent Collection Discovery**: Enable users to discover artworks through natural language queries, visual similarity, and thematic connections.

3. **Personalized Museum Experiences**: Create tailored virtual and physical museum visits based on user preferences and educational goals.

4. **Cultural Context Integration**: Connect artworks to their historical, social, and cultural backgrounds for deeper understanding.

5. **Multi-Modal Art Analysis**: Combine textual information with image analysis for comprehensive artwork understanding.

### **Challenges**

1. **Data Quality and Consistency**: Managing artwork metadata from multiple sources with varying quality and completeness standards.

2. **Cultural Sensitivity**: Ensuring respectful representation of diverse cultures and avoiding Western-centric interpretations of global art.

3. **Image Rights Management**: Navigating copyright restrictions and usage rights for artwork images across different institutions.

4. **Scalability Across Collections**: Integrating data from thousands of museums and galleries with different cataloging systems.

5. **Educational Adaptation**: Tailoring content complexity and presentation style to different age groups and expertise levels.

### **Potential Impact**

- **Democratized Art Education**: Making high-quality art education accessible globally regardless of geographic location
- **Enhanced Museum Engagement**: Increasing visitor engagement through personalized, interactive experiences
- **Research Facilitation**: Supporting art historians and researchers with comprehensive, searchable databases
- **Cultural Preservation**: Digitally preserving and sharing cultural heritage for future generations
- **Cross-Cultural Understanding**: Promoting global cultural awareness through accessible art exploration

    def get_artworks(self, page: int = 1, size: int = 100) -> List[Dict[str, Any]]:
        """Fetch artworks from Harvard Art Museums API"""
        try:
            url = f"{self.base_url}/object"
            params = {
                'apikey': self.api_key,
                'page': page,
                'size': size,
                'hasimage': 1  # Only get objects with images
            }
            
            response = requests.get(url, params=params)
            if response.status_code == 200:
                data = response.json()
                return data.get('records', [])
            else:
                print(f"Error fetching Harvard artworks: {response.status_code}")
                return []
                
        except Exception as e:
            print(f"Error in Harvard API call: {e}")
            return []

                return response.json()
            return None

            print(f"Error fetching artist info: {e}")
            return None

            return None

                    if artwork and artwork.get('primaryImage'):
                        artworks.append(artwork)
                    time.sleep(0.1)  # Rate limiting
                
                return artworks

class MuseumDataAggregator:
    """Aggregates data from multiple museum sources"""
    
    def __init__(self):
        self.harvard_collector = HarvardArtMuseumCollector()
        self.met_collector = MetMuseumCollector()
        self.wiki_collector = WikiArtCollector()
    
    def collect_comprehensive_data(self) -> Dict[str, List[Dict[str, Any]]]:
        """Collect data from all sources"""
        all_data = {
            'artworks': [],
            'artists': [],
            'exhibitions': []
        }
        
        # Collect from WikiArt (sample data)
        wiki_artists = self.wiki_collector.get_artists()
        wiki_artworks = self.wiki_collector.get_artworks()
        
        all_data['artists'].extend(wiki_artists)
        all_data['artworks'].extend(wiki_artworks)
        
        # Collect from Harvard (if API key available)
        if Config.HARVARD_ART_API_KEY:
            harvard_artworks = self.harvard_collector.get_artworks(size=50)
            for hw_artwork in harvard_artworks:
                standardized_artwork = self._standardize_harvard_artwork(hw_artwork)
                all_data['artworks'].append(standardized_artwork)
        
        # Collect from Met Museum
        try:
            met_search_results = self.met_collector.search_artworks("painting")
            for met_artwork in met_search_results[:10]:  # Limit for demo
                standardized_artwork = self._standardize_met_artwork(met_artwork)
                all_data['artworks'].append(standardized_artwork)
        except Exception as e:
            print(f"Error collecting Met data: {e}")
        
        return all_data
    
    def _standardize_harvard_artwork(self, hw_data: Dict[str, Any]) -> Dict[str, Any]:
        """Standardize Harvard artwork data"""
        return {
            'id': f"harvard_{hw_data.get('id', '')}",

import chromadb
from chromadb.config import Settings

class ArtHistoryVectorStore:
    def __init__(self):
        self.embeddings = OpenAIEmbeddings(
            openai_api_key=Config.OPENAI_API_KEY,
            model=Config.EMBEDDING_MODEL
        )
        
        self.client = chromadb.PersistentClient(
            path=Config.CHROMA_PERSIST_DIRECTORY,
            settings=Settings(anonymized_telemetry=False)
        )
        
        # Initialize collections
        self.artworks_collection = self._get_or_create_collection("artworks")
        self.artists_collection = self._get_or_create_collection("artists")
        self.exhibitions_collection = self._get_or_create_collection("exhibitions")
        self.cultural_contexts_collection = self._get_or_create_collection("cultural_contexts")
    
    def _get_or_create_collection(self, name: str):
        try:
            return self.client.get_collection(name)
        except:
            return self.client.create_collection(name)
    
    def add_artwork(self, artwork_data: Dict[str, Any]):
        """Add artwork to vector store"""
        try:
            searchable_text = f"""
            Artwork: {artwork_data.get('title', '')}
            Artist: {artwork_data.get('artist', '')}
            Date: {artwork_data.get('date_created', '')}
            Medium: {artwork_data.get('medium', '')}
            Museum: {artwork_data.get('museum', '')}
            Description: {artwork_data.get('description', '')}
            Culture: {artwork_data.get('culture', '')}
            Period: {artwork_data.get('period', '')}
            Dimensions: {artwork_data.get('dimensions', '')}
            """
            
            embedding = self.embeddings.embed_query(searchable_text)
            
            self.artworks_collection.add(
                documents=[searchable_text],
                embeddings=[embedding],
                metadatas=[artwork_data],
                ids=[artwork_data['id']]
            )
            
        except Exception as e:
            print(f"Error adding artwork to vector store: {e}")
    
    def add_artist(self, artist_data: Dict[str, Any]):
        """Add artist to vector store"""
        try:
            searchable_text = f"""
            Artist: {artist_data.get('name', '')}
            Birth Year: {artist_data.get('birth_year', '')}
            Death Year: {artist_data.get('death_year', '')}
            Nationality: {', '.join(artist_data.get('nationality', []))}
            Biography: {artist_data.get('biography', '')}
            Art Movements: {', '.join(artist_data.get('art_movements', []))}
            Major Works: {', '.join(artist_data.get('major_works', []))}
            Influences: {', '.join(artist_data.get('influences', []))}
            """
            
            embedding = self.embeddings.embed_query(searchable_text)
            
            self.artists_collection.add(
                documents=[searchable_text],
                embeddings=[embedding],
                metadatas=[artist_data],
                ids=[artist_data['id']]
            )
            
        except Exception as e:
            print(f"Error adding artist to vector store: {e}")
    
    def add_cultural_context(self, context_data: Dict[str, Any]):
        """Add cultural context information"""
        try:
            searchable_text = f"""
            Period: {context_data.get('period', '')}
            Movement: {context_data.get('movement', '')}
            Location: {context_data.get('location', '')}
            Historical Context: {context_data.get('historical_context', '')}
            Key Characteristics: {context_data.get('characteristics', '')}
            Important Artists: {', '.join(context_data.get('important_artists', []))}
            Cultural Significance: {context_data.get('cultural_significance', '')}
            """
            
            embedding = self.embeddings.embed_query(searchable_text)
            
            self.cultural_contexts_collection.add(
                documents=[searchable_text],
                embeddings=[embedding],
                metadatas=[context_data],
                ids=[context_data.get('id', f"context_{len(searchable_text)}")]
            )
            
        except Exception as e:
            print(f"Error adding cultural context: {e}")
    
    def search_similar(self, query: str, collection_name: str, n_results: int = 5) -> List[Dict]:
        """Search for similar items in specified collection"""
        try:
            collection = getattr(self, f"{collection_name}_collection")
            query_embedding = self.embeddings.embed_query(query)
            
            results = collection.query(
                query_embeddings=[query_embedding],
                n_results=n_results
            )
            
            return [
                {
                    'metadata': metadata,
                    'document': document,
                    'distance': distance
                }
                for metadata, document, distance in zip(
                    results['metadatas'][0],
                    results['documents'][0],
                    results['distances'][0]
                )
            ]
            
        except Exception as e:
            print(f"Error searching vector store: {e}")
            return []
    
    def find_similar_artworks(self, artwork_id: str, n_results: int = 5) -> List[Dict]:
        """Find artworks similar to a given artwork"""
        try:
            # Get the artwork details first
            artwork_results = self.artworks_collection.get(ids=[artwork_id])
            if not artwork_results['metadatas']:
                return []
            
            artwork_data = artwork_results['metadatas'][0]
            
            # Create query based on artwork characteristics
            query = f"{artwork_data.get('artist', '')} {artwork_data.get('period', '')} {artwork_data.get('medium', '')}"
            
            # Search for similar artworks
            similar = self.search_similar(query, "artworks", n_results + 1)
            
            # Filter out the original artwork
            return [item for item in similar if item['metadata'].get('id') != artwork_id][:n_results]
            
        except Exception as e:
            print(f"Error finding similar artworks: {e}")
            return []
    
    def get_artwork_by_artist(self, artist_name: str, n_results: int = 10) -> List[Dict]:
        """Get artworks by a specific artist"""
        try:
            results = self.search_similar(f"artist {artist_name}", "artworks", n_results)
            # Filter to only include works by the specified artist
            return [item for item in results if artist_name.lower() in item['metadata'].get('artist', '').lower()]
        except Exception as e:
            print(f"Error getting artworks by artist: {e}")
            return []

    def generate_artwork_analysis(self, artwork_id: str) -> Dict[str, Any]:

            if not artwork_results['metadatas']:
                return {'error': 'Artwork not found'}
            
            artwork_data = artwork_results['metadatas'][0]
            
            # Get related information
            artist_info = self._get_artist_context(artwork_data.get('artist', ''))
            cultural_context = self._get_cultural_context(artwork_data)
            similar_works = self.vector_store.find_similar_artworks(artwork_id, 3)

            analysis = self._generate_analysis_with_rag(

                'cultural_context': cultural_context,

            # Prepare context

    def create_virtual_tour(self, theme: str, difficulty: str = "intermediate") -> Dict[str, Any]:
        """Create a themed virtual tour"""
        try:
            # Search for artworks related to the theme
            relevant_artworks = self.vector_store.search_similar(theme, "artworks", 8)
            
            if not relevant_artworks:
                return {'error': 'No artworks found for this theme'}
            
            # Generate tour narrative
            tour_narrative = self._generate_tour_narrative(theme, relevant_artworks, difficulty)
            
            tour = {
                'id': f"tour_{theme.replace(' ', '_')}",
                'title': f"Virtual Tour: {theme.title()}",
                'description': f"Explore {theme} through carefully selected artworks",
                'theme': theme,
                'difficulty_level': difficulty,
                'artworks': [work['metadata'] for work in relevant_artworks],
                'narrative': tour_narrative,
                'duration_minutes': len(relevant_artworks) * 5,  # 5 minutes per artwork
                'stops': self._create_tour_stops(relevant_artworks)
            }
            
            return tour
            
        except Exception as e:
            return {'error': f'Error creating virtual tour: {e}'}
    
    def _generate_tour_narrative(self, theme: str, artworks: List[Dict], difficulty: str) -> str:
        """Generate narrative for virtual tour"""
        try:
            context = f"Theme: {theme}\nDifficulty: {difficulty}\n"
            context += "Selected Artworks:\n"
            
            for artwork in artworks:
                metadata = artwork['metadata']
                context += f"- {metadata.get('title', 'Unknown')} by {metadata.get('artist', 'Unknown')}\n"
            
            prompt = f"""
            Create an engaging introduction and narrative for a virtual museum tour with the following details:
            
            {context}
            
            The narrative should:
            1. Introduce the theme and its significance
            2. Explain what visitors will discover
            3. Set appropriate expectations for the {difficulty} difficulty level
            4. Create excitement about the artworks they'll see
            5. Be approximately 150-200 words
            
            Make it welcoming and educational while matching the difficulty level.
            """
            
            messages = [
                SystemMessage(content="You are an expert museum guide who creates engaging tour narratives."),
                HumanMessage(content=prompt)
            ]
            
            response = self.chat_model(messages)
            return response.content
            
        except Exception as e:
            return f"Welcome to this virtual tour exploring {theme}. Let's discover some amazing artworks together!"
    
    def _create_tour_stops(self, artworks: List[Dict]) -> List[Dict[str, Any]]:
        """Create detailed stops for the virtual tour"""
        stops = []
        
        for i, artwork in enumerate(artworks, 1):
            metadata = artwork['metadata']
            
            # Generate brief commentary for each stop
            stop = {
                'stop_number': i,
                'artwork': metadata,
                'duration_minutes': 5,
                'key_points': self._generate_key_points(metadata),
                'questions_to_consider': self._generate_reflection_questions(metadata)
            }
            
            stops.append(stop)
        
        return stops
    
    def _generate_key_points(self, artwork_data: Dict) -> List[str]:
        """Generate key points about an artwork"""
        key_points = []
        
        if artwork_data.get('artist'):
            key_points.append(f"Created by {artwork_data['artist']}")
        
        if artwork_data.get('date_created'):
            key_points.append(f"Made in {artwork_data['date_created']}")
        
        if artwork_data.get('medium'):
            key_points.append(f"Medium: {artwork_data['medium']}")
        
        if artwork_data.get('period'):
            key_points.append(f"Part of the {artwork_data['period']} period")
        
        return key_points[:4]  # Limit to 4 key points
    
    def _generate_reflection_questions(self, artwork_data: Dict) -> List[str]:
        """Generate reflection questions for artwork"""
        questions = [
            "What emotions does this artwork evoke for you?",
            "What details do you notice first when looking at this piece?",
            "How might this artwork have been received when it was first created?"
        ]
        
        if artwork_data.get('period'):
            questions.append(f"How does this artwork reflect the values of the {artwork_data['period']} period?")
        
        return questions[:3]  # Limit to 3 questions

    def answer_visitor_question(self, question: str, context_type: str = "general") -> str:
